# Eventos Cidade

Projeto simples de uma plataforma para visualização e gerenciamento de eventos, permitindo criar, editar e excluir eventos em diferentes cidades.

## Tecnologias Utilizadas

- **Frontend:** React, React Router, Axios
- **Backend:** Node.js, Express, Mongoose
- **Banco de Dados:** MongoDB (utilizando `mongodb-memory-server` para desenvolvimento)

## Instalação

1.  Clone o repositório:
    ```bash
    git clone https://github.com/brunokitice/eventos-cidade.git
    ```

2.  Entre na pasta do projeto:
    ```bash
    cd eventos-cidade
    ```

3.  Instale as dependências do frontend (projeto principal):
    ```bash
    npm install
    ```

4.  Instale as dependências do backend:
    ```bash
    cd backend
    npm install
    cd ..
    ```

## Execução

Para iniciar a aplicação (frontend e backend juntos), rode o seguinte comando a partir da pasta raiz do projeto:

```bash
npm start
```

A aplicação estará disponível em `http://localhost:3000`.
A API estará rodando em `http://localhost:5000`.
